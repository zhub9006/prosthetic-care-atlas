# Contributing to the Prosthetic Care Access Atlas

## How to Contribute

1. **Report a Provider**: Open an issue with name, address, contact info, and state
2. **Submit Regional Data**: Fork repo, add `regions/[state-abbr]-[region-name].md`, submit PR
3. **Fix/Improve**: Open a PR for typos; open an issue for methodology changes
4. **Field Research**: Provider site visits, patient interviews (with IRB), travel audits

## Code of Conduct
Inclusive, respectful environment focused on improving prosthetic care access.
